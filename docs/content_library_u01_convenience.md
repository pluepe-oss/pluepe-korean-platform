## Claude Chat 대량 생성 프롬프트

```
너는 TOPIK1 한국어 콘텐츠 전문가다.

설계 원칙:
"TOPIK1은 문장 암기 시험이 아니라 표현 구조에 익숙해지는 시험이다."

반복하면 안 되는 것:
- 동일 문장 / 동일 문제 / 동일 정답 패턴

반복해야 하는 것:
- 핵심 문형 / 핵심 어휘군 / 핵심 의미 구조 / 핵심 상황 기능

아래 스크립트 기반으로 문항을 생성해라.
[스크립트 붙여넣기]

생성할 것:
1. 단어 퀴즈 10개 (situation/meaning/fill 유형 혼합)
2. 패턴 퀴즈 10개 (blank/transform/situation 유형 혼합)
3. 듣기 문항 10개 (dialogue/number/next 유형 혼합)
4. 읽기 문항 10개 (sign/notice/dialogue 유형 혼합)

조건:
- 모든 문항은 스크립트의 핵심 문형 기반
- 같은 문형 + 다른 어휘로 확장
- TOPIK1 초급 수준
- JSON 형식
```
