> 작성일: 2026.04.22 | 무료체험관 포함 최종 확정판

## 확정 결정사항

| 항목 | 결정 |
|---|---|
| 무료체험관 | 메인 최상단 CTA / 카드등록 필수 / 유닛 1개 전체 경험 |
| 상품 단위 | 시험 선택 → 언어 선택 순서 |
| 언어 변경 | 최초 1회 / 이후 변경 불가 / 어드민 변경 가능 |
| 유닛 잠금 | 순차 강제 → 전체 완료 후 자유 |
| 모의고사 | 유닛 전체 완료 후 순차 오픈 (3회 → 5회) |
| TOPIK2/EPS | 구조 재사용 + 콘텐츠 업그레이드 |

## 1. 공개 영역 (비로그인)

```
/                               홈 (랜딩 페이지)
│
│   ┌─────────────────────────────────────────┐
│   │  🎯 메인 CTA (최상단 고정)               │
│   │  [7일 무료로 시작하기] ← 가장 큰 버튼   │
│   │  [플랜 보기]            ← 보조 버튼      │
│   └─────────────────────────────────────────┘
│
│   - 서비스 소개 ("하루 20분, 4주 완성 TOPIK1")
│   - 학습 흐름 소개 (단어→패턴→세션→테스트→AI)
│   - 상품 미리보기 카드
│   - 수강 후기
│
├── /trial                      🆓 무료체험관 (핵심 메뉴)
│   │
│   ├── Step 1: 회원가입
│   │   - 이메일 / Google / 카카오
│   │
│   ├── Step 2: 시험 선택
│   │   - TOPIK1 / TOPIK2 / EPS-TOPIK
│   │
│   ├── Step 3: 언어 선택
│   │   - TOPIK1/2: 베트남어 / 영어 / 중국어
│   │   - EPS: 베트남어 / 영어 / 인도네시아어
│   │   - ⚠️ "선택 후 변경 불가" 알럿
│   │
│   ├── Step 4: 카드 정보 입력 (Stripe)
│   │   - "7일 무료 · 이후 자동 결제" 안내
│   │   - "언제든 해지 가능" 안내

│       - 유닛 1개만 접근 가능
│       - 완료 or 7일 경과 → 구독 전환 유도 모달
│
├── /pricing                    상품 & 가격
│   - TOPIK1 패키지 (3개 언어)
│   - TOPIK2 패키지 (3개 언어)
│   - EPS-TOPIK 패키지 (3개 언어)
│   - 번들 패키지
│   - B2B 학원 좌석제
│   - [7일 무료로 시작하기] CTA
│
├── /auth                       로그인
└── /auth/signup                회원가입

## 2. 유료 구매 플로우 (결제 후)

## 3. 학습자 영역 (로그인 후 — Student)

```
/my                             마이페이지 (학습 컨트롤 타워)
│
├── [Today]                     오늘 할 것 (Phase 자동 추천)
│   ├── Phase 1 (Day 1~12)     유닛 이어하기 버튼
│   ├── Phase 2 (Day 13~20)    약점 복습 추천
│   └── Phase 3 (Day 21~28)    모의고사 시작
│
├── [Learning Flow]             현재 단계 위치
│   - Phase 1 / 2 / 3 진행 표시
│   - 예상 완료일
│
├── [Progress]                  학습 진도
│   - 유닛 완료 N / 전체
│   - 연속 학습일 🔥 streak
│   - 유닛 카드 목록
│     ├── ✅ 완료 (자유 열람 가능)
│     ├── 🔵 진행 중
│     └── 🔒 잠금 (순차 강제)
│
├── [Weakness]                  약점 분석
│   - 단어 / 패턴 / 듣기 / 읽기 점수
│   - 약점 기반 복습 추천
│
├── [Exam]                      시험 현황
│   - 🔒 유닛 전체 완료 전 잠금
│   - 완료 후 순차 오픈 (3회 → 5회)
│   - 점수 추이 차트
│
└── [Account]                   계정/구독
    - 구독 플랜 + 만료일
    - 선택 언어 표시 (변경 불가)
    - Stripe 구독 관리 포털

## 6-1. 무료체험 마이페이지 상세

> 유료 마이페이지와 **완전히 동일한 디자인**
> 유닛 1개 제외 모든 기능 🔒 잠금 + 기능 설명 표시

```
/my (무료체험 상태)

┌─────────────────────────────────────────────┐
│  🔵 Today — 오늘 할 것                       │
│  [유닛 1 이어하기 →]   ← 유일하게 활성       │
│  예상 시간: 25분                             │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  🔵 Progress — 학습 진도                     │
│  유닛 1 진도 표시 (활성)                     │
│  유닛 2~ 카드: 🔒 잠금 표시                  │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  🔒 Weakness — 약점 분석                     │
│  단어 / 패턴 / 듣기 / 읽기 영역별 점수를    │
│  자동으로 분석해드려요.                      │
│  학습 데이터가 쌓일수록 더 정확해집니다.     │
│  [구독하고 확인하기 →]                       │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  🔒 Exam — 모의고사                          │
│  TOPIK 실전과 동일한 문제 구성으로           │
│  실력을 점검하고 합격 가능성을 확인하세요.   │
│  TOPIK1 기준 듣기 30문항 + 읽기 40문항      │
│  [구독하고 시험 보기 →]                      │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  🔒 Vocab — 단어 암기                        │
│  틀린 단어를 자동으로 모아                   │
│  간격 반복으로 오래 기억하게 해드려요.       │
│  [구독하고 시작하기 →]                       │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐
│  🔵 Account — 구독 관리                      │
│  현재: 무료체험 중 (D-7까지)                 │
│  선택 언어: 베트남어                         │
│  [지금 구독하기 →]  ← 오렌지 CTA 버튼       │
│  체험 종료 후 자동 결제됩니다.               │
└─────────────────────────────────────────────┘
```

### 잠금 처리 원칙
| 구분 | 처리 방식 |
|---|---|
| UI | 유료와 동일한 디자인 |
| 잠금 표시 | 🔒 아이콘 + 흐림(blur) 처리 |
| 설명 텍스트 | 기능 소개 1~2줄 반드시 표시 |
| CTA | "구독하고 [기능명] →" 버튼 |
| 목적 | "돈을 내면 이 기능을 받는다" 인지 |
