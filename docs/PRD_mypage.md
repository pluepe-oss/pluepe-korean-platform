### 섹션 2 — Today (오늘 할 것)

**가장 중요한 섹션. 최상단 배치.**

#### Phase 자동 추천 로직

```
학습 시작일 기준 현재 Day 계산

Day 1~12   → Phase 1: 유닛 학습
Day 13~20  → Phase 2: 약점 복습
Day 21~28  → Phase 3: 모의고사
```

#### Phase별 Today 화면

| Phase | 표시 내용 | 버튼 |
|---|---|---|
| Phase 1 | 현재 진행 중 유닛명 + 진도율 + 예상시간 | [이어하기 →] |
| Phase 2 | 약점 영역 + 복습 추천 항목 | [복습 시작 →] |
| Phase 3 | 모의고사 회차 + 최근 점수 | [모의고사 시작 →] |

#### 무료체험자 Today
- 유닛 1만 표시
- [이어하기 →] 버튼 활성

### 섹션 3 — Progress (학습 진도)

| 요소 | 내용 |
|---|---|
| 전체 완료율 | 완료 유닛 수 / 전체 유닛 수 |
| 연속 학습일 | 🔥 N일 streak |
| 유닛 카드 목록 | 유닛별 상태 카드 |

#### 유닛 카드 상태

| 상태 | 표시 | 클릭 |
|---|---|---|
| 완료 ✅ | 초록 체크 + 진도 100% | 자유 열람 가능 |
| 진행 중 🔵 | 진도바 N% | 이어하기 |
| 잠금 🔒 | 흐림 처리 | 이전 유닛 완료 후 해제 |

#### 무료체험자 Progress
- 유닛 1 카드만 활성
- 유닛 2~ 카드: 🔒 잠금 + "구독하면 12개 유닛 전체를 학습할 수 있어요"
