## 4. 섹션별 상세 기능

### 4-1. 세션 플레이어 (STEP 1~5)

| STEP | 이름 | 내용 | 완료 조건 |
|---|---|---|---|
| STEP 1 | 보기 | 전체 영상 시청 + 이해 퀴즈 1개 (3지선다) | 영상 시청 AND 퀴즈 정답 |
| STEP 2 | 빈칸 연습 | 영상 재시청 + 빈칸 3문제 (3지선다) | 영상 시청 AND 3문제 완료 |
| STEP 3 | 따라 말하기 | 문장 3개 자가 체크 (✓ 아이콘) | 3개 체크 완료 |
| STEP 4 | 단어 익히기 | 단어 카드 6개 플립 + 퀴즈 1문제 | 퀴즈 완료 |
| STEP 5 | 복습하기 | 핵심 표현 3개 카드 (클릭 시 뜻+예문) | 1개 이상 클릭 |

**영상 정책:**
- Bunny.net Stream iframe embed
- 영상 완료 감지: Player.js SDK (ended 이벤트 + 95% 재생 fallback)
- seek 스킵 방지: 미완료 STEP에서 앞으로 건너뛰면 완료 초기화
- 자동재생 없음 (autoplay=false)

**STEP 탭 표시:**
- 완료 전: 숫자 원문자 ①②③④⑤
- 완료 후: ✓ 표시

## 5. 진도 관리

### 섹션 완료 조건 요약

| 섹션 | 완료 조건 |
|---|---|
| session | STEP 1~5 전부 완료 |
| words | 퀴즈 3문제 답변 |
| patterns | 빈칸 퀴즈 3문제 답변 |
| test | 3문제 답변 |
| ai | AI 1회 이상 호출 |

### 진도율 계산
```
섹션당 20% × 완료 섹션 수 = 유닛 진도율
예: session + words 완료 = 40%
```

### 완료 배너
- 5섹션 모두 완료 시에만 표시
- [다음 유닛 →] 버튼: /unit/2 이동
- [유닛 완료] 버튼: /my 이동
