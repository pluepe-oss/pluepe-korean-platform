## 1. 제품 개요

### 1.1 핵심 가치
```
"TOPIK 합격할 때까지 플루페가 함께"
강의 + AI 챗봇 + 모의고사 + 단어외우기 = 합격 패키지
```

### 1.2 타겟
- B2C: 한국어를 배우려는 외국인 개별 학습자
- B2B: 한국어 학원 (인도네시아·중국·캄보디아·베트남·한국)

## 2. 언어별 상품 구성

### 2.1 상품 종류

| 상품 | 자막/설명 언어 | 대상 국가 |
|------|--------------|----------|
| 한국어-베트남어 | 베트남어 | 베트남 |
| 한국어-영어 | 영어 | 글로벌 |
| 한국어-중국어 | 중국어 | 중국 |
| 한국어-인도네시아어 | 인도네시아어 | 인도네시아 |

### 2.2 각 상품 포함 콘텐츠

```
강의
  오픈 시: TOPIK 1 / TOPIK 2 / EPS-TOPIK 각 4강 = 총 12강
  추가: Master 콘솔에서 언어별/시험별 자율 운영
        (베트남어 3강, 영어 1강 등 자유롭게)
  상한 없음 / order_index 기준 자동 정렬

단어외우기
  강의(topic)별 핵심단어(core) + 응용단어(extended) 연동
  SRS 간격 반복 복습
  AI 질문: 응용단어(extended)만 가능
  핵심단어(core): AI 질문 불가 (영상에서 학습한 단어)

모의고사
  총 12개 (TOPIK 1: 4개 / TOPIK 2: 4개 / EPS-TOPIK: 4개)
  순차 오픈 정책 적용

AI 챗봇
  한국어 학습 Q&A / 오답 해설
  사용 횟수 제한 정책 적용
```

## 3. 모의고사 오픈 정책

### 3.1 B2C 월간 구독자
```
가입 1개월차  → 1개 오픈
가입 2개월차  → 2개 오픈
...
가입 12개월차 → 12개 전체 오픈
효과: 12개월 구독 유지 동기 부여
```

### 3.2 B2C 연간 구독자
```
가입 즉시     → 6개 오픈
가입 6개월 후 → 나머지 6개 오픈 (전체 12개)
효과: 연간 결제 유도 + 6개월 재방문 동기
```

### 3.3 B2B 학원
```
계약 즉시 → 12개 전체 오픈
이유: 학원 수업 커리큘럼 자유롭게 구성 필요
     연간 계약으로 이미 묶여있어 제약 불필요
```

## 4. AI 챗봇 사용 정책

### 4.1 챗봇 호출 포인트
```
① 단어외우기 → 응용단어 "AI에게 질문하기" (/api/vocab/chat)
② AI 학습 챗봇 메뉴 → 자유 한국어 Q&A (/api/chat)
→ 두 채널 합산 카운트
```

### 4.2 B2C 개인 정책
```
하루 제한: 20회 (단어질문 + 일반챗봇 합산)
초기화: 매일 자정
초과 시: "오늘 AI 질문 횟수를 모두 사용했습니다. 내일 다시 이용해주세요."
```

### 4.3 B2B 학원 정책
```
ID별 월 제한: 20회 (단어질문 + 일반챗봇 합산)
초기화: 매월 1일

좌석별 총량:
  10석: ID당 20회 × 10명 = 월 200회
  20석: ID당 20회 × 20명 = 월 400회
  30석: ID당 20회 × 30명 = 월 600회

초과 시: "이번 달 AI 질문 횟수를 모두 사용했습니다."
원장 어드민: ID별 잔여 횟수 확인 가능
관리: platform_settings에서 조정 가능
```

### 4.4 영업 데모 계정
```
하루 5회 제한
```

### 4.5 공통 규칙
```
핵심단어(core): AI 질문 불가
응용단어(extended): AI 질문 가능
단어 챗봇: 해당 단어 관련 질문만 답변
무관한 질문: "이 기능은 단어 학습 전용입니다."
```

## 5. 가격 정책

### 5.1 B2C 개인 구독

| 플랜 | 가격 | 비고 |
|------|------|------|
| 월간 | $9.99/월 | 언제든 취소 가능 |
| 연간 | $89/년 | 7일 이내 환불 / 이후 환불 없음 |

```
연간 할인율: 약 26% ($9.99 × 12 = $119.88 → $89)
취소 후 구독 기간 끝까지 이용 가능
```

### 5.2 B2B 학원 라이선스 (연간 일시불)

| 좌석수 | 연간 가격 | 월 환산 | 1인당/년 | 1인당/월 | B2C 대비 할인 | 10석 대비 추가 할인 |
|--------|----------|---------|---------|---------|-------------|----------------|
| 10석   | $300/년  | $25/월  | $30/년  | $2.50/월 | 66% 할인    | 기준            |
| 20석   | $500/년  | $42/월  | $25/년  | $2.08/월 | 72% 할인    | 17% 추가 할인   |
| 30석+  | $700/년  | $58/월  | $23/년  | $1.94/월 | 74% 할인    | 22% 추가 할인   |

```
B2C 연간 기준: $89/년 = $7.42/월 (할인율 비교 기준)

구독제 아닌 라이선스제 (자동 갱신 없음)
연간 일시불 결제
만료 30일 전 재계약 알림 발송
만료 시 자동 접근 차단
ID 교체: 학원장이 어드민에서 자유롭게 관리
동시 재생: 좌석 수만큼만 허용 (Cloudflare Signed URL 제한)
```

### 5.5 크로스 언어 접근 정책
```
언어 = 상품 단위 (B2C / B2B 동일 적용)

한국어-베트남어 구독/라이선스
→ 베트남어 강의/단어외우기/시험보기만 접근 가능
→ 영어/중국어/인도네시아어 콘텐츠 접근 불가 ❌

추가 언어 필요 시 → 별도 상품 구매
B2B 학원도 동일:
→ 베트남어 라이선스 학원 → 베트남어 콘텐츠만
→ 타 언어 원하면 별도 계약
```

### 5.3 영업 데모 계정
```
account_type = 'demo'
각 시험별 3강씩 (TOPIK1: 3강 / TOPIK2: 3강 / EPS: 3강)
모의고사: 샘플 5문항만
AI 챗봇: 하루 5회
만료일 설정 가능 (7일/14일/30일)
Master 콘솔에서 발급/관리
```

### 5.4 무료체험 (별도 메뉴)
```
/free-trial 별도 페이지 (유료 구독과 분리)

제공 범위:
  강의: 각 시험별 1강 맛보기
  단어: 핵심단어만 (is_free=true)
  모의고사: 샘플 5문항
  AI 챗봇: 불가 (구독자 전용)
```

## 6. B2C vs B2B 상품 비교표

| 항목 | B2C 월간 | B2C 연간 | B2B 학원 |
|------|---------|---------|---------|
| 가격 | $9.99/월 | $89/년 | $300~/년 |
| 결제 방식 | 구독 자동결제 | 구독 자동결제 | 연간 일시불 |
| 강의 | 전체 | 전체 | 전체 |
| 단어외우기 | 전체 | 전체 | 전체 |
| 모의고사 | 월 1개 순차 | 즉시 6개+6개 | 즉시 12개 전체 |
| AI 챗봇 | 하루 20회 | 하루 20회 | ID당 월 20회 |
| 어드민 | 없음 | 없음 | 원장 어드민 포함 |
| 동시 재생 | 1기기 | 1기기 | 좌석 수만큼 |
| 환불 | 다음달 취소 | 7일 이내 | 없음 |

